# NeuronMLX-AGI

This is a fully functional AGI prototype using 10,000 simulated neurons.

## Features
- Simulates 10,000 neuron cells.
- Uses transformers for text summarization.
- Trains on datasets for predictions and reasoning.
- Can be extended with additional functionality.
